Variable Declearation:
Constant:
int age = 13
inline assignment like that is a constant
Mutable:
int :age = 13 or
int age
:age = 13

Datatypes
int, float, complex, str, bin, dict, @seq, set, tup
int a = 5
float b = 1.5
complex c = 12j
str name = "Rex"
bin10 = 01010101
dict details = {int key: int value}
set d = <{5, 4, 3}> or :{5, 4, 3}
tup = (int age, str name)

Protocols
prot greet_user(None) -> None:
  output "Hello, user"

Control flow
if int 5 < int 20 and "a" != "b":
  output "Mehh"

Loops
for :i in 0..10 step 2:
  output(i)

while True:
  output "Hello"
  stop
